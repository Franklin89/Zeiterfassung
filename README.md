# Zeiterfassung

## Features

- Benutzer CRUD
- Projekt CRUD
- Zeiterfassen mit Tätigkeit und Projekt (Eintrag Editieren / Löschen)
- Anmelden nur mit Benutzername
- Auswertung der Arbeitsstunden

### Mögliche Erweiterung

- Offline Modus
- Anmeldung erweitern mit Passwort
- Grafik für die Auswertung
